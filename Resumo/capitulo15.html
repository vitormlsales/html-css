Seletores Personalizados

Usando id com css

            Em HTML id = em CSS #
            Em HTML class = em CSS .

Aprendemos anteriormente a usar CSS nos seletores, criando configurações de estilo para cada seletor. Mas quando quisermos diferenciarmos um <h1> de um <h1>, por exemplo. Isso é possivel criando um seletor personalizado.

Em html utilizamos dentro da tag o id="", ficando assim: <h1 id="">. E quando estivermos configurando o CSS representaremos assim h1#.

As diferenças entre id e class

id é utilizado para apenas uma tag por página. Funciona se você utilizar em multiplas tag, mas estará dando o uso indevido.

Class é o mais indicado para o uso em multiplas tags.

id em html = # em css

class em html = . em css

Pseudo-classes em CSS

Pseudo-classes é relacionado a um estado do elemento, se está ativo, marcado, etc...

Representa-se uma pseudo-classe com dois pontos (:), fazendo sua configuração em CSS:

            h1:hover{configurações}

A pseudo-classe hover é para o estado de estar passando o mouse por cima. Então as configurações que você fizer utilizando ele só aparecerão ao passar o mouse em cima.

Pseudo-classes conhecidas por mim:

  • :hover
  • :visited
  • :active

Pseudo-elementos em CSS

Pseudo-elementos mexem diretamente no conteúdo periférico do elemento. Representação em CSS por dois pontos (::).

Pseudo elementos conhecidos por mim:

  • ::after
  • ::before

Observações:

  • # = id
  • . = class
  • : = pseudo-class
  • :: = pseudo-element
  • > = children
